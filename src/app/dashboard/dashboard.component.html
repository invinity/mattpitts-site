Hello, and thank you for visiting. This site is about me, a career software and engineering professional, and acts as a small demonstration of some of my experience.

About This Site

This site is meant to demonstrate some of my skills and experience throughout the entire software lifecycle.

The site is an Angular application that is built, tested, packaged and deployed into a hosting environment completely with CI/CD (Continuous Integration/Continuous Delivery) automation. The source code for the project is hosted as a public github repository.

Realtime Resume

This site also contains a component to render my personal resume as a web page using live information from my LinkedIn profile. This is accomplished using the LinkedIn API and it allows myself, or anyone else, to view a nicely styled resume that is always up-to-date with my LinkedIn profile. This demonstrates my experience working with APIs along with OAuth authentication.

Fully Automated Build, Test, Deploy

This application's CI/CD is automated using GitHub Actions. The steps are defined in a workflows file contained within the project's source and are executed each time there is a push to the main branch of the project. This means that any change to the project that is pushed or merged onto the project's default git branch is automatically executed in the CI/CD pipeline and the application is re-deployed into is hosting environment.

Hosting

This application is hosted as a static website within an AWS (Amazon Web Services) S3 bucket. The bucket is configured to allow serving of the files within it as a static website.
