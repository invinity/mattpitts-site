About This Site

This site is meant to demonstrate some of my skills and experience throughout the entire software lifecycle.

The site itself is the result of an Angular application that is built, tested, packaged and deployed into a hosting environment completely with CI/CD (Continuous Integration/Continuous Delivery) automation. The source code for the project is hosted as a public github repository here

Realtime Rendered Resume

This site also contains a component to render my personal resume as a web page using live information from my LinkedIn profile. This is accomplished using another project, written in Golang, that provides a simple API to retrieve my Linkedin profile as JSON. This it allows myself, or anyone else, to view a formatted resume that is always up-to-date with my LinkedIn profile.

Fully Automated Build, Test, Deploy

This Angular project backing this site has fully-automated CI/CD using GitHub Actions. The steps are defined in a workflows file contained within the project's source and are executed each time there is a push to the main branch of the project. This means that any change to the project that is pushed or merged onto the project's default git branch is automatically executed in the CI/CD pipeline and the application is re-deployed into is hosting environment.

In addition, the Golang project providing the Linkedin profile API also has fully-automated CI/CD. But instead of Github Actions, this project is using Google Cloud Build to automatically pickup any commits to the Github repository and perform CI/CD automation. This includes the construction of a Docker images for the Golang project and the deployment of this image into Google Cloud Run for hosting.

Hosting of this site

The Angular application for this site is hosted as a static website within an AWS (Amazon Web Services) S3 bucket. The bucket holds the site's constructed Angular output files, while an AWS CloudFront distribution is setup to serve the actual files via HTTP requests, as well as providing Transport Layer Security (TLS) for the site.

For the Golang project providing the Linkedin profile data for myself, it is deployed as a Docker container into Google Cloud Run
